Report

Introduction

The website is a movie review website that informs, educate, give an depth analysis on movies and latest news surreound celebraties through articles and podcasts. First, I took inspiration from other similar websites since using a structure similar to other websites creates a sense of familiarity for my visitors. The website has a flat structure that ensures that pages are just a few clicks away from each other which saves users time and easy for search engine crawlers.The Website navigation is a user friendly and allows for easy navigation on the site.

Inspiration

  1. I got inspirations from websites like: filminquiry.com, lwlies.com and flickreel.com
  2. Since this project requires a magazine or newsletter, I was inspired to create a movie review site because it's easy to get access to trailers/videos, pictures, and audio files.
  3. I'm a fun of movies so this project inspired me to create a movie review website and I look forward to build upon it someday

Accessibility

  1. I have provided alt text or description for non-text content. So when a screen reader reads an image or graphic, it will read the image's alternative ("alt") text.
  2. By using descriptive titles for every page, screen readers announce the page title (the "title" element in the HTML markup) when first loading a Web page
  3. I also provided headings to organize the page content. One common way to skim a page with a screen reader is to jump from heading to heading. Users can hear an overview of the page's key information, then backtrack to read the parts they are most interested in

Usability

  1. I decided to add more videos to enhance the content of the site by grabbing the attention of the users who visit to the site.
  2. Another aspect I considered is the website responsiveness. Responsive design addresses the idea of making websites usable by different devices, by larger display devices such as laptops and smaller display devices such as mobile phones
  3. Visitors come to website with specific goals in mind and not just come and admire your web design skills. Clarity is the core of website usability and having a web site design that distracts or confuses visitors will drive them away and unlikely to come back.

Learning

  1. Before this project, I didn't know anything about bootstrap. I had to learn bootstrap and it really helped me a lot throughout the project
  2. I also had learnt how to create a logo for my project and other tools for creating wireframes.
  3. There were little things I didn't know before this project like how to import css, import fonts from the internet, other css functions, and how to add javascript to your code.

Evaluation I

Being new to web site development, I think the layout of my website with the use of bootstrap played a huge success for my project. And the use of carousel to automatically slides tweets were fine to implement.

Evaluation II

One aspect of my project that could be improved is where I have the Trending News. I've wanted each news to show up on the page automatically and then slides for the next news to show up but I couldn't fix that so I had to provide a next and previous buttons to control the news. I think I will do more research on that and hopefull get the best solution on how to fix the problem.

Resources

Resources I used in my code are Bootstrap, Figma, and fontAwesome
